# Gün Projesi: Styled Alışveriş Sitesi

Geçen gün spor hocan için yaptığın alışveriş sitesi beğenildi. Ama, senin içinde sinmedi ve daha iyi bir UI için kolları sıvadın.

ilk olarak hızlıca klasik html ve css'i kullandın. Ama gördün ki bu iş gittikçe büyüyecek, daha iyi bir yöntem kullanmak istedin. Aklına 'stled-components' kullanmak geldi.

Önceden hazırladığın Products component'ini styled component olarak hazırlayacaksın.

- İpucu: class isimlerini component isimleri olarak kullanabilirsin.
  örneğin:
  .price-tag yerine PriceTag
  .description yerine Description

- İpucu: class kullanılmamış tag'ler için de product'ın property ismini kullanbilirsin.
  örneğin:
  product.image yerine Image
  product.title yerine Title

- İpucu: hover pseudo-class'ını styled component'in içinde yazabilirsin.

- İpucu: styled bir component'e prop ileterek dinamik stil kuralları oluşturabilirsin.
